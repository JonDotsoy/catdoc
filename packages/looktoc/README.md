# Looktoc

Definition to write custom toc file.

**Example:**

```json
{
  "$schema": "https://unpkg.com/@catdoc/looktoc@0.3.5/schema.json",
  "items": [
    {
      "type": "item",
      "title": "Overview",
      "uri": "docs/overview.md"
    },
    {
      "type": "item",
      "title": "API",
      "uri": "docs/reference/api.md"
    }
...
```
